# 🎯 Quick Reference - New Features

## Upload Profile Image

| Step | Action | Result |
|------|--------|--------|
| 1 | Click **"Edit Profile"** button | Modal opens |
| 2 | Click **"Choose Image"** | File picker opens |
| 3 | Select image from computer | Preview shows |
| 4 | Click **"Save Profile"** | Image saved & displayed |

**Note:** Image is optional - can skip if you want

---

## Search & Add Platform

| Step | Action | Result |
|------|--------|--------|
| 1 | Click **"+ Add New App"** | Modal opens with search |
| 2 | Start typing platform name | Results filter in real-time |
| 3 | Click platform in results | Platform gets selected |
| 4 | See platform highlighted | (Confirmation it's selected) |
| 5 | Enter profile URL | (Your GitHub URL, etc.) |
| 6 | Click **"Add Link"** | Link appears on profile |

## Troubleshooting 🔧

### Image not showing?
- Make sure you clicked "Save Profile"
- Check file size (should be < 5MB)
- Try JPG instead of PNG

### Can't find platform?
- Try different search term
- "git" finds GitHub, GitLab, GitBook
- "link" finds LinkedIn
- Or type full name

### Link not saving?
- Make sure platform is selected (highlighted in blue)
- Make sure URL starts with `https://`
- Check browser console (F12) for errors

### Data disappeared?
- Check browser tabs - using same browser?
- Check if "Private/Incognito" mode (doesn't save data)
- Try clearing browser cache
